# Page Loader

[![Actions Status](https://github.com/zluuba/python-pytest-testing-project-79/workflows/hexlet-check/badge.svg)](https://github.com/zluuba/python-pytest-testing-project-79/actions)

Page Loader is a library that knows how to download pages over the network and save them to the local drive. <br>
**This is a study project**, so it's not too useful for humanity, but useful for me. <br>
If you want to try it, you can use the instructions below. <br>
How package works: [click]().

### Requirements

- [python](https://www.python.org/), version 3.9 or higher
- [poetry](https://python-poetry.org/docs/#installation), version 1.0.0 or higher

### Installation

Clone this repo or download it with pip:
```ch
git clone https://github.com/zluuba/page-loader.git
```
```ch
pip install --user git+https://github.com/zluuba/page-loader.git
```

Install package and dependencies:
```ch
cd page-loader                # don't forget cd to downloaded package dir
make install
make build
make package-install
```
